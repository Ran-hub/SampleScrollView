# SampleScrollView
In iPhone app development, the iOS scroll view is generally used to view content that can’t fit entirely in a single screen. This UIScrollView is one of the most useful and versatile control.

In this [iOS app tutorial](https://www.spaceotechnologies.com/ios-scroll-view-example-tutorial/), we’ve build a sample app to demonstrate how to use iOS scroll view control for iPhone app development to display content larger than a single screen efficiently.
